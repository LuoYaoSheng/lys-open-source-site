# lys Open Source Site

这是个人开源系统的公开网站仓库，不只是站点骨架。

它承担三类角色：

- 开源项目入口
- 学习路径入口
- 定期文章与公开更新入口

如果用户不想先看仓库结构，这个站点应该先告诉他们：

1. 从哪里开始体验
2. 从哪里开始学习
3. 哪些项目最值得先看

## 技术栈

- Astro
- TypeScript
- 静态输出

## 页面

- `/`
- `/learn`
- `/articles`
- `/projects`
- `/weekly`
- `/roadmap`
- `/about`

## 面向谁

- 想学习 BLE 与 IoT 的学习者
- 想通过真实项目来理解工程实现的人
- 想先体验，再决定要不要读源码的人
